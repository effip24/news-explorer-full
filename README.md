### run the project locally:


#### first, run the server:

```
  git clone https://github.com/effip24/news-explorer-full.git
```

```
  cd news-explorer-full/backend
```

```
  npm install
```

```
  npm run start
```
#### second, run the client:

 
```
cd ../frontend
```
  
```
npm install
```

```
  npm run start
```
